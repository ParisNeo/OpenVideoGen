How to Use:

  • Enter a creative prompt in the "Create a Video" section.
  • Select a model or use the default, set frames, and click "Generate Video."
  • Copy the Job ID and paste it in "Job Status" to track progress.
  • Click "Check" to start monitoring—progress updates automatically!
  • Download your video once completed.

Tips:

  • Progress updates every 2 seconds during processing.
  • Visit GitHub for more details.
